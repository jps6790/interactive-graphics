Project 4:

Interactive Graphics on the Web

Hello, and welcome to my humble exploration of canvas-based drawing on the web! If you haven't yet checked out my simple yet amusing applications, simply click on the images above.

This site was primarily created to test canvas functionality in a real web environment, using the p5.js library for efficient rendering and user interaction. I faced some early challenges regarding p5's draw() function, as it required values from global variables that had not yet been initialized by the time draw() was called. My solution was to link draw() to a boolean that became true only once the window.onload event had occurred. I also attempted to pass information about each tool into the document via JavaScript, but I found that presenting one application while retrieving and displaying data on the others was particularly challenging, so I resolved to simply passing in the name and description of the currently selected tool. On another note, I thought it might be a good idea to reinforce clicking on a tool's thumbnail by implementing a simple CSS color transition for its hover state.
